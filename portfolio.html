왜 CDC를 선택했냐면: DB Replication은 스키마 변경이 어렵고, Dual Write는 앱 코드가 복잡해지고, MQ 기반은 메시지 유실 위험이 있고, ETL 배치는 실시간이 안 됩니다. CDC는 DB 레벨에서 변경을 캡처하니까 앱 코드 수정 없이 실시간 동기화가 되거든요. AWS DMS가 RDS 호환성이 가장 높아서 선택했습니다.
